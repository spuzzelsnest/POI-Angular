# POI-Angular

A remake in angular of a project once started in 2007. This is a POI app based on [OpenStreetMap](https://www.openstreetmap.org/#map=8/50.510/4.475) and [Leaflet plugin](https://leafletjs.com/)

# The APP

## Install

- Install postgress
- run npm i
- run migrations
- run seeders

## Run the App

- run backend
- run app
